# Impact Queue — 5-minute TCM drill

Use this with new teammates before go-live.

## Daily flow (3 steps)

1. **Open** `/impact` — read **Suggested now** at the top.
2. Tap **Do it** — complete the action in the sidebar (schedule, quote, call, etc.).
3. Close the sidebar and move to the next card (or the next **Hard action** pill).

## Board columns (left → right)

| Column | Meaning |
|--------|---------|
| Inbox | New leads — complete Impact profile and schedule |
| Tour scheduled | Tour booked — confirm, reschedule, or start |
| On tour today | Today's tours — mark tour done |
| Quote sent | Follow up / negotiate / book |
| Booked | Check-in & handover |

## Drawer flow

The lead drawer now opens only the allowed work step:

Impact → Tour → Post-tour → Quote → Check-in

TCMs cannot jump ahead from the step rail. Finish the current step to unlock the next one.

## Filters

- Only **one** filter chip active at a time (Hot, Overdue, Tour today, etc.).
- Use **Reset filters** if the board looks empty by mistake.
- **Search** is for finding a name — it does not change what **Do it** does.

## Focus inventory

- **Manage focus** → pin 3–5 properties you are pushing today.
- Search Property Hub + ops inventory; pins save in your browser.

## Keyboard (optional)

- `J` — next lead in stack order  
- `K` — previous lead  
- `Enter` — open selected lead  

## If the board is empty

- Check the red **Can't load leads** banner — retry or ask ops.
- Clear search and tap **Reset filters**.

## Managers

- Use **All TCMs** in the header dropdown.
- **Escalations by TCM** shows who needs help.
